# Medical-RAG  
A Retrieval-Augmented Generation (RAG) based application designed for medical use-cases such as policy lookup, claim assistance, document Q&A, and knowledge retrieval.  
This system combines semantic search (FAISS vectorstore) with an LLM (Ollama + Llama3.2) to provide accurate and context-aware responses.
